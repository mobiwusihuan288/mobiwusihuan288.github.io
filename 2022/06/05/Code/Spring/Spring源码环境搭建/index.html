# 关联 GitHub

注意下文补充

补充：针对下载的压缩包重新关联 GitHub, fork 源码请无视

由于 spring 项目自带 .gitignore 文件

spring-framework\spring-aop\src\main\java\org\springframework\aop

目录下的 target 会被忽略

解决：分两步 [git add .]

1. 先将 .gitignore 文件剪切到其他位置，执行 git add .

2. 还原 .gitignore 文件，再次执行 git add .
